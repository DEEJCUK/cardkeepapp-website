Updated color scheme, improved multi-term search functionality, and enhanced UI improvements

New Features

Updated Color Scheme

Fresh new visual identity matching design patterns with modern, accessible colors that enhance readability and user experience across all screens.

Improved Multi-term Search

Enhanced search functionality that handles multiple search terms more intelligently, making it easier to find specific cards using natural language queries.

Bug Fixes

Button Contrast in Dark Mode: Improved button visibility and contrast ratios in dark mode for better accessibility and readability.
Crop Modal Layout: Fixed layout issues in the photo crop modal that were affecting user experience during photo editing.

UI Improvements

Repositioned Buttons in Photo Crop Modal: Optimized button placement in the crop modal for better usability and more intuitive photo editing workflow.
Enhanced Text Readability: Improved text contrast and typography across all themes to ensure excellent readability in both light and dark modes.
